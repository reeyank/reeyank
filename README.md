### 🚀 What I'm building

I like turning half-formed ideas into working products fast — usually with **Python + TypeScript**, an **agentic/LLM** twist, and a deadline. A few things I've shipped:

| Project | What it is |
| --- | --- |
| 🏛️ **[Hall of Us](https://github.com/reeyank/hall-of-us-backend)** | A museum-inspired social memories app built at HackGT — with a modular, **agentic, LangChain-inspired** FastAPI backend. |
| 🤖 **[ios-test](https://github.com/reeyank/yc-hackathon-integration-tests)** | One-command **autonomous integration testing** for React Native iOS apps. An AI agent reads your source, drives the Simulator like a human QA tester, and emits a real test suite. |
| 📚 **[githubDocGen](https://github.com/reeyank/githubDocGen)** | Point it at *any* public GitHub repo and it generates a full documentation website — analyzes the codebase and builds the docs for you. |
| 🏀 **[nba_ml](https://github.com/reeyank/nba_ml)** | Predicts NBA player stats using **XGBoost** models, trainable from scratch. |
| 📊 **[websiteAnalytics](https://github.com/reeyank/websiteAnalytics)** | A drop-in analytics tracker + **FastAPI** backend — pageviews, clicks, custom events, user identification. |
| 🖨️ **[Calendar → Receipt Printer](https://github.com/reeyank/GoogleCalendarToRecieptPrinter)** | A fun hardware hack that prints your Google Calendar events to a USB thermal printer via webhooks. |
